# FUTURE_DS_02
Internship Project – Social Media Campaign Performance Tracker using Power BI. Analyzed ad metrics like CTR, CPC and ROI using Facebook/Instagram data. Built interactive dashboards to visualize campaign success and provided actionable insights. #MarketingAnalytics #PowerBI #FutureInterns
📈 Social Media Campaign Performance Tracker
📌 Internship Task 2 – Future Interns (Data Science & Analytics)
This project was completed as part of Task 2 in the Future Interns Data Science & Analytics Internship. The objective was to analyze and visualize paid social media campaign data using Power BI, helping businesses understand digital ad performance and make data-driven marketing decisions.

🎯 Objective
To evaluate marketing campaign effectiveness by answering:

📊 Which ads or posts had the highest engagement?

🖱️ What were the Click-Through Rate (CTR) and Cost Per Click (CPC)?

💰 How much Return on Investment (ROI) was generated?

📈 What can be improved for future campaigns?

📁 Dataset
Name: Facebook / Instagram Ads Performance Dataset

Format: CSV

Source: Kaggle - Facebook Ads Performance


🧼 Data Cleaning & Preprocessing
Using Power BI and Excel:

Removed duplicates and blank entries

Converted date fields to proper formats

Cleaned cost and metric columns (CPC, CTR, ROI, etc.)

Normalized categorical data (age groups, devices, regions)

🛠 Tools & Technologies
Power BI Desktop – Interactive dashboarding and data modeling

Microsoft Excel / Google Sheets – Initial cleaning & quick aggregations

Looker Studio / Canva – For design and alternative visual reporting

📊 Dashboard Features
🧮 Campaign Overview: Impressions, Reach, Clicks, Spend

📈 CTR, CPC, ROI KPIs

🏆 Top-Performing Ads or Creatives

🌍 Performance by Region / Device / Age Group

🔍 Interactive Filters for in-depth exploration

📝 Recommendations for future marketing strategy

💡 Key Insights
Ads with video creatives had a higher CTR than static posts

Mobile users accounted for over 70% of total reach

ROI was strongest in age group 25–34

High CPC observed in underperforming regions — consider retargeting strategies
